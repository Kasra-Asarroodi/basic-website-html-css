# HTML & CSS Basics Website

This project is a basic website created to understand the fundamentals of HTML and CSS.  
It includes multiple pages and demonstrates simple layout, navigation, and styling techniques.

## Features
- Multiple web pages (e.g. Home, Admin, Adoption, Application)
- Basic HTML structure
- CSS styling for layout and design
- Navigation between pages

## Purpose
The purpose of this project is to practice and improve foundational web development skills using HTML and CSS.
